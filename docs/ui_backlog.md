# ZePlay — UI Backlog & Future Improvements

This backlog documents planned frontend features, enhancements, and UX polish items scheduled for post-Sprint 3 releases.

---

## 1. Catalog & Discovery Enhancements
- [ ] **Advanced Filter & Sort**: Filter by genre multi-select, release year range, maturity rating, and sort by popularity / release date.
- [ ] **Interactive Recommendations**: AI-driven "Because you watched..." horizontal carousels with dynamic matching scores.
- [ ] **Trailer Preview Modal**: Auto-play muted video trailer on card hover after 1.5s delay.
- [ ] **Watchlist / My List**: One-click "Add to My List" state management with optimistic UI updates.

## 2. Video Player Polish & Advanced Controls
- [ ] **Custom HTML5 Video Player Overlay**: Replace native controls with branded glassmorphic controls (play/pause, timeline scrubber, volume slider, fullscreen, picture-in-picture).
- [ ] **HLS Quality Selector UI**: Auto / 1080p / 720p / 480p resolution switcher dropdown during playback.
- [ ] **Audio Track & Subtitle Switcher**: Multi-language audio and SRT/VTT caption selection menu.
- [ ] **Skip Intro / Next Episode Overlay**: Dynamic timed overlay buttons for tv series/episodes.
- [ ] **Resume Playback**: Save watching progress (`timestamp_seconds`) per profile and display "Continue Watching" progress bar on poster cards.

## 3. Account & Profile Management
- [ ] **Profile Avatar Selector Grid**: Rich collection of branded avatars / character icons for profile customization.
- [ ] **PIN Protection for Profiles**: 4-digit PIN lock for adult profiles.
- [ ] **Billing & Subscription Dashboard**: Plan upgrade/downgrade UI, payment method management, invoice history.

## 4. Accessibility & Performance
- [ ] **Full Keyboard Navigation**: Focus traps and shortcut keys (`Space` = Play/Pause, `F` = Fullscreen, `M` = Mute, `Left/Right` = 10s Skip).
- [ ] **Skeleton Loaders**: Custom shimmering skeletons for media grid items during async fetching.
- [ ] **PWA & Offline Support**: Service worker caching for shell and offline catalog browsing.
